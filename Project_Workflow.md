# 🔄 Project Workflow

3. Cleaned and transformed data using Power Query
4. Created Date Table and data model (Star Schema)
5. Developed DAX measures for KPIs and time intelligence
6. Designed interactive dashboards with slicers and tooltips
7. Published Power BI file via GitHub Releases due to file size
